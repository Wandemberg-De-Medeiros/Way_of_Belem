# Way of Belém
Way of Belém é ser um serious game socioeducativo onde o jogador é colocado no lugar do prefeito da cidade de Belém do Pará e precisa tomar decisões para manter a cidade em ordem

Neste cenário, o jogador terá que enfrentar escolhas que vão desafiar sua capacidade de gestão, lidando com situações que o obrigará a fazer escolhas difíceis e duvidosas para conseguir manter a harmonia dos poderes que governam esta cidade
